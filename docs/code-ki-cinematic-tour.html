O que essa consulta faz?

SELECT name, date, year
FROM users
WHERE name LIKE 'A%';
A Seleciona nome, data e ano da tabela users, apenas nomes que comecam com A.
B Cria uma nova tabela chamada users.
C Apaga usuarios que nao comecam com A.
D Atualiza o ano dos usuarios com A no final.
Resultado: SELECT escolhe colunas, FROM indica a tabela, WHERE filtra, LIKE 'A%' busca nomes iniciados por A.
